Companion OSC API

/location/<page>/<row>/<column>/press — press and release (no args)
/location/<page>/<row>/<column>/down — press and hold
/location/<page>/<row>/<column>/up — release
/location/<page>/<row>/<column>/rotate-left — encoder rotate left
/location/<page>/<row>/<column>/rotate-right — encoder rotate right
 
Example — trigger Page 1, Row 0, Column 2:
/location/1/0/2/press
 
Row and column are 0-based. On a standard 8×4 grid:
columns 0–7, rows 0–3

Enable the OSC listener in Companion under Settings → Remote Control → OSC. Default port is 12321.

VirtualBox USB Passthrough Checklist

  1. Install VirtualBox Extension Pack (same version as VirtualBox)
  2. Add your Linux user to the vboxusers group:
    sudo usermod -aG vboxusers $USER then log out/in
  3. In VM Settings → USB: enable USB 2.0 (EHCI) or USB 3.0 controller
  4. Add a USB filter: Vendor ID + Product ID only (find with lsusb on host)
  5. Plug in the PerfectCue receiver after the VM is running — it will be captured automatically
  6. Inside the VM, confirm it appears: python3 perfectcue_bridge.py --list-devices
  7. Add the bridge user to the input group:
    sudo usermod -aG input bridge

Quick Install

# Clone / copy files to VM
sudo cp -r perfectcue-bridge/ /opt/perfectcue-bridge/
 
# Run installer
cd /opt/perfectcue-bridge && sudo bash install.sh
 
# Check status
sudo systemctl status perfectcue-bridge
sudo systemctl status perfectcue-web
 
# Web UI available at:
http://<vm-ip>:8080
